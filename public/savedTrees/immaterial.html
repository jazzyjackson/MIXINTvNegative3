asdflkjasdfkjhsadflj this is editable. but hitting 'return' kinda screws things up because contentEditable does crazy things. But you won't notice that until you save the page and refresh. But anyway. Shift click and drag pans the whole page. The terminals and editors floating above the page can be moved by click and dragging their headers. Anyone else who has the same url as you will have their actions synchronized. The terminals evaluate any javascript, so the page can be freely manipulated or have features collaboratively coded and appended to the document on the fly. Javascript is a language that can be used to modify itself (object prototypes and methods can be overwritten by functions inside of runtime) so it seemed appropriate to build a website that could be used to overwrite itself.
